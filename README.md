## 🌐 Deploy to Vercel (Step by Step)

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "Initial portfolio commit"
git branch -M main
git remote add origin https://github.com/ferdous440/liakat-portfolio.git
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **"Add New Project"**
3. Select your `liakat-portfolio` repository
4. Click **"Deploy"** — done! ✅

Your site will be live at: `https://liakat-portfolio.vercel.app`
